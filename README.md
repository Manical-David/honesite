# 个人作品集网站

基于 Vue 3 + TypeScript + Tailwind CSS 构建的现代化个人作品集网站。

## 🌟 特性

- 💫 优雅的页面过渡动画
- 🌓 深色模式支持
- 📱 响应式设计
- 🛠 实用工具集（JSON 格式化等）
- 📝 博客系统
- 🎨 技能展示
- 📬 联系表单

## 🔧 技术栈

- **前端框架：** Vue 3 (组合式 API)
- **类型支持：** TypeScript
- **样式解决方案：** Tailwind CSS
- **构建工具：** Vite
- **代码规范：** ESLint + Prettier
- **版本控制：** Git
